About This Project

This project is basically a complete study of MySQL as an open-source software.
At first, I used MySQL normally in projects without thinking much, but while doing this audit I understood how much effort and philosophy is behind it.

In this project, I explored:

Why MySQL was created
How open-source licensing  actually works
Its working on Linux systems
Its role in the open-source ecosystem
Comparison with proprietary software like SQL Server

Along with theory, I also created shell scripts to show practical understanding of Linux and scripting.

----------------------------This repository includes------------------------------------:

5 shell scripts (.sh files)
Concepts of Linux commands and scripting
Practical implementation related to system, logs, and open-source ideas


1.System Identity Report

This  shows basic system details like:

Kernel version
Username
Linux distribution
Uptime
Date and time

It uses simple commands like uname, whoami, date and variables.

2.FOSS Package Inspector

This  checks whether MySQL is installed or not.
It also shows version details and gives a small idea about the software.

It automatically detects whether the system is using apt or rpm, so it works on different Linux systems.

3.Disk and Permission Auditor

This  checks:

Important system directories
Permissions and ownership
Disk usage

It also specifically checks MySQL-related directories like:
/var/lib/mysql, /etc/mysql, etc.

4.Log File Analyzer

This  reads a log file and:

Counts total lines
Finds keyword occurrences (like "error")
Shows last few matching lines

5.Open Source Manifesto Generator

This is an interactive script.
It asks a few questions and creates a small personal statement about open source.

It then saves it into a file with your username.
